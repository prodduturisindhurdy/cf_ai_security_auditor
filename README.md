# cf_ai_security_auditor

This is an AI-powered security auditor built on Cloudflare Workers using the Llama 3.3 model. It is designed to identify potential vulnerabilities in backend code snippets.

### Live URL
[https://noisy-darkness-0484.prodduturisindhu66.workers.dev/](https://noisy-darkness-0484.prodduturisindhu66.workers.dev/)

### How it Works
- Built with **Cloudflare Workers AI**.
- Uses the **Llama 3.3-70b-instruct** model.
- Accepts code via POST requests and returns a security audit in JSON format.
